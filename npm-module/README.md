**Getting started**

  * Clone this repo locally
  * Open a commandline to the ``` ./npm-module ``` directory in this repo
  * Run ``` npm install ``
  * To run the examples, execute the following command: ``` npm run runExample ```. **NOTE**: This will clean and rebuild the dist folder
